## Culmen Length - Culmen Depth Relation


Now, try to visualize the general relation between Culmen Length and Culmen Depth of palmer penguins via a general scatter plot:

```python
import seaborn as sns
sns.scatterplot(
    data = penguins,
    x = 'Culmen Length (mm)',
    y = 'Culmen Depth (mm)'
)
```
